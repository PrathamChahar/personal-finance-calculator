# Personal Finance Calculator 💰

A Python command-line tool that helps users track their monthly income, 
expenses and savings — and gives personalized financial advice.

## Features
- Calculates total monthly expenses
- Calculates monthly savings automatically
- Shows savings percentage
- Gives personalized financial advice based on saving habits

## How to Run
1. Make sure Python is installed
2. Clone this repository
3. Run the file:
```
python finance.py
```
